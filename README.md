# Hermes AI Desktop

Hermes AI Desktop is an Electron-based desktop application for AI-powered productivity. 

## Features
- Chat with AI models
- Workspace management
- Model monitoring
- API key management

## Getting Started

1. Clone the repository:
   ```sh
   git clone https://github.com/silva2kand/hermes-ai-desktop.git
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Run the app:
   ```sh
   npm run dev
   ```
